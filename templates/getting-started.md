# Welcome to Dulcet

Start typing to edit this document. Your changes are saved in your browser until you save to GitHub.

## Getting Started

- Open the **Command Palette** (Ctrl+Shift+P) to access all commands
- Use **NoteThink: Auto View** to switch how the right panel renders
- Press **Ctrl+S** to save your work to a GitHub repository
